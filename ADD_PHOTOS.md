# How to Add Photos to the Gallery

1. Drop new photo files into SylvesterPalm/images/gallery/
2. Open Claude Code and paste this prompt:
   "Scan /SylvesterPalm/images/gallery/ for any new image files not yet in the gallery section of index.html, add them to the gallery with the existing lightbox + Get a Quote button functionality, then push to GitHub with commit message 'Add new gallery photos'."
3. Done. Photos appear on the live site in 30 seconds.

Supported formats: .jpg, .jpeg, .png, .webp, .HEIC
